# Nextcloud User Files Backup

[English version](https://github.com/alekslegkih/hassio-addons/blob/main/nc_user_files_backup/README.md)

Аддон Home Assistant для автоматического резервного копирования пользовательских
файлов Nextcloud на внешнее USB-хранилище.

Резервное копирование выполняется по расписанию и поддерживает инкрементальный
режим с использованием `rsync`.  
Аддон предназначен для работы с заранее смонтированными внешними накопителями.

## Возможности

- Автоматическая работа с внешними USB-дисками
- Инкрементное резервное копирование с использованием `rsync`
- Планирование заданий в формате cron
- Управление питанием через умные выключатели (опционально)
- Валидация конфигурации перед запуском
- Тестовый режим выполнения
- Поддержка уведомлений (Telegram, мобильные уведомления и др.)

## Требования

- Home Assistant OS
- Внешний накопитель, смонтированный с использованием правил `udev`
- Доступ к корневой файловой системе хоста

## Важная информация

> [!CAUTION]
> Для работы аддона требуется доступ к корневой файловой системе Home Assistant OS.  
> Аддон вмешивается в работу операционной системы, управляемой супервизором.
